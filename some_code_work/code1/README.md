对于n个有序数据点，其相邻间隔构成n-1个差值序列。定义平均间隔avg_gap = (最大值 - 最小值)/(n-1)，根据鸽巢原理可推导： 
1. 均匀分布时最大间隔等于平均间隔
2. 当存在某个间隔小于平均间隔时，必然存在其他间隔大于平均间隔

算法实现: 
1. 边界确定：获取数据集的最小值min和最大值max
2. 分桶策略：
· 创建n-1个桶，每个桶的区间宽度为avg_gap
· 第k号桶范围：[min + k×avg_gap, min + (k+1)×avg_gap)
3. 数据映射：
· 通过公式⌊(x-min)/avg_gap⌋确定元素所属桶编号
· 每个桶仅需记录区间内的最小值和最大值
4. 间隔计算：
· 遍历相邻非空桶，计算后桶最小值与前桶最大值的差值
· 所有相邻桶差值中的最大值即为全局最大间隔

复杂度分析： 
· 时间复杂度：O(n)（仅需线性遍历数据及桶结构）
· 空间复杂度：O(n)（需存储n-1个桶的边界信息）
