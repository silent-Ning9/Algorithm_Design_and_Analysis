算法步骤
分解阶段：

将 $2^k \times 2^k$ 的棋盘划分为四个 $2^{k-1} \times 2^{k-1}$ 的子棋盘

特殊方格必定位于其中一个子棋盘

解决阶段：

如果子棋盘包含特殊方格，则递归处理该子棋盘

对于不含特殊方格的三个子棋盘，在它们相邻的角上放置一个L型骨牌，使这三个子棋盘各有一个"伪特殊方格"

递归处理这四个子棋盘

合并阶段：

当棋盘大小为 $1 \times 1$ 时，递归终止

所有子棋盘覆盖完成后，整个棋盘也被完全覆盖


时间复杂度
递归关系：$T(k) = 4T(k-1) + O(1)$

解递归方程得：$T(k) = O(4^k)$

由于棋盘有 $4^k$ 个方格，时间复杂度为 $O(n)$，其中 $n$ 是棋盘方格总数

空间复杂度
递归栈深度：$O(k)$

棋盘存储：$O(4^k)$

总空间复杂度：$O(4^k)$
